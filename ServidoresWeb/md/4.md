# Activar el módulo “wsgi” para permitir la ejecución de aplicaciones Python

Para activar el módulo `wsgi`, ejecutaremos el comando:

``` cmd
sudo apt-get install libapache2-mod-wsgi-py3
```

![](img/34.png)

``` cmd
sudo a2enmod wsgi
```
